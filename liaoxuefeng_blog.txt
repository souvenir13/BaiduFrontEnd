1.== 比较 自动转换数据类型再进行比较。=== 比较 不会自动转换数据类型。
  始终坚持使用 ===
2.浮点数的比较：要计算两个浮点数是否相等，只能计算他们的差的绝对值，是否小于某个阈值。
  Math.abs(1/3 - (1 - 2/3))<0.0000001; //true

5.遍历map array set

    // element: 指向当前元素的值

6.arguments - 用于函数内部，获得函数传入的所有参数。
由于js函数可以传入任意个数的参数，如果只想获取前两个参数，可以利用rest获取多余参数：
  function foo(a,b,...rest){}
  foo(1,2,3,4,5);//传入的参数先绑定a,b;多传入的参数以数组形式交给rest
7.全局变量会绑定到window上，不同的JavaScript文件如果使用了相同的全局变量就会产生冲突，减少冲突的方法将自己定义的所有变量和函数都绑定到一个全局变量中：
  var PINKCHAN = {};//唯一的全局变量
  PINKCHAN.name = 'myapp';//其他变量
  PINKCHAN.version = 1.0;
  PINKCHAN.foo = function(){};//其他函数
